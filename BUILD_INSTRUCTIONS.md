# How to Build the Executable

## Prerequisites

1. Install Python dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Make sure `h3mtxt.exe` is in the project root directory

## Building the EXE

### Using the build script (Recommended)

Run the batch script:

```cmd
.\build_exe.bat
```

## Output

The executable will be created in the `dist/` folder as `MapGenerator.exe`
